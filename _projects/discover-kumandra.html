--- layout: project title: Discover Kumandra excerpt: Microsite promoting Raya and the Last Dragon. description:

Venture into the world of Kumandra to discover the lands and meet the characters from Raya and the Last Dragon. See visual development artwork and learn about the research behind the film.

I was the sole developer on this project and also helped create animations and UI enhancements.

This site was built using Jekyll, SASS, GSAP, ScrollTrigger, and JavaScript. It also uses CloudCannon as a CMS to allow for easy adjustments and updates.
